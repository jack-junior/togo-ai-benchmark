Instructions pour les annotateurs

Merci de lire ces consignes avant d’annoter :

  • Fournissez un identifiant annotateur unique (p. ex. ANNO01).
  • Donnez une note 1–5 pour chaque critère, 5 étant le meilleur.
  • Justifiez brièvement chaque note (60–300 caractères).
  • Si vous n’êtes pas sûr.e d’un élément, cochez la note la plus prudente et commentez.
  • Ne partagez pas vos identifiants ou le token d’accès.

Si une protection par token est activée, fournissez-le via le paramètre `?token=...` dans l’URL ou via l’entête `X-Eval-Token`.
